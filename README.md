# PySpark Examples

This project contains the Apache Spark Python examples copied from the local Spark 4.0.2 binary distribution.

## Runtime

- Python interpreter: `/opt/anaconda3/bin/python`
- Spark home: `/Users/Fitzgerald/Applications/spark/current`
- Java home: `/Users/Fitzgerald/Applications/java/current/Contents/Home`

## Run

Example command:

```bash
JAVA_HOME=/Users/Fitzgerald/Applications/java/current/Contents/Home \
SPARK_HOME=/Users/Fitzgerald/Applications/spark/current \
/opt/anaconda3/bin/python examples/pi.py 10
```
